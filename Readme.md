## Description
Spend Sense is a lightweight, intuitive expense tracking application designed specifically for students to manage their daily cash flow. 

## Setup & Run Instructions
1. Extract the `WebProject_[YourRollNo].zip` file.
2. Open the extracted folder in VS Code or your terminal.
3. Start the JSON Server mock backend by running:
   `npx json-server --watch db.json`
4. In a separate terminal or using Live Server, open `index.html` in your web browser.

## Features
* **User Panel:** Full submission form with 5 fields, inline validation without alerts, category filtering, and immediate DOM re-rendering.
* **Admin Panel:** Separate interface with distinct styling, comprehensive statistics (Total Count, Total Amount, Average), and full Edit/Delete functionality.
* **Bonus:** Fully styled using Bootstrap 5 for clean layout and responsiveness.
* **Tech Stack:** HTML5, Bootstrap 5, Vanilla JS (No React/Frameworks), JSON Server Backend.
